lesson 62 : MicroPythonで成績を平均化してソートする関数を作成する
=============================================================================

このチュートリアルでは、Raspberry Pi Pico WでMicroPythonの関数を使って成績を平均化し、ソートする方法を説明します。

**タスク概要**:
 - ユーザーから成績を入力する。
 - 成績を表示する。
 - 成績の平均を計算する。
 - 最高点と最低点を見つける。
 - 成績を高い順にソートする。

**主なステップ**:
 - 関数を定義し呼び出す。
 - 変数と配列を関数に渡す。
 - ソートアルゴリズムを実装する。

**宿題**:
 - 成績を入力し、平均を計算し、最高点と最低点を見つけ、高い順にソートするプログラムを、事前に構築されたライブラリを使わずに関数を使って書く。
